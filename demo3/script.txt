# メソッドをサブコマンドに、fields をオプションにマップすると、
こういう便利さも有りますよ、という話でした
# 例えばこんなシンプルな OO Modulino でも、
メソッド名の補完が効きます
# :Doc や fields のサポートは私の Base クラスを継承するのが手早いです
が、それが無くてもメソッド名の補完は使えます
# オプションの解説は $FIELDS{オプション名}->{doc} から
取り出しています
# オプション名も一覧表示と補完が可能です。
オプションは fields (%FIELDS)から取り出しています。
# このようにメソッドと解説の一覧が簡単に閲覧できれば、
プログラムを書いてしばらく経った後でも全容が把握しやすくなります
# メソッド名の後の解説は、メソッドに付けた
 :Doc() という attribute から取り出しています
# _perl_oo_modulino をインストールすると、
メソッド名の一覧表示と補完が出来ます
# この .pm ファイルは任意のメソッドをサブコマンドとして
呼び出せる、OO な Modulino です
# zsh で .pm のメソッド名を補完する話
_perl_oo_modulino という completer のデモをします
